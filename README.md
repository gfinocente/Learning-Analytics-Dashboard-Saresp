# 📊 Dashboard de Desempenho Escolar

Projeto de análise de dados educacionais com foco em gerar insights acionáveis para a tomada de decisão pedagógica.

---

## 🎯 Objetivo

Transformar dados de desempenho escolar em informações estratégicas para apoiar gestores e professores na identificação de dificuldades e oportunidades de melhoria.

---

## 🔒 Privacidade dos dados

Os dados foram anonimizados para garantir a privacidade dos alunos, em conformidade com a Lei Geral de Proteção de Dados (LGPD).

---

## 📊 Funcionalidades

* Visualização da distribuição dos níveis de proficiência
* Comparação de desempenho entre turmas
* Identificação de disciplinas com maior dificuldade
* Geração automática de insights
* Dashboard interativo com filtros

---

## 🛠️ Tecnologias

* Python
* Pandas
* Streamlit
* Plotly

---

## 📈 Principais insights

* Concentração significativa de alunos nos níveis mais baixos de proficiência
* Diferença de desempenho entre turmas
* Identificação de disciplinas críticas para intervenção

---

## 🚀 Como executar

```bash
pip install streamlit pandas openpyxl plotly
streamlit run dashboard_interativo.py
```

---

## 📌 Sobre o projeto

Este projeto demonstra como a análise de dados pode ser aplicada na educação para gerar valor real e apoiar decisões baseadas em evidências.
